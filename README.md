# 🍽 Food Care

Android와 FastAPI 기반의 식자재 관리 및 맞춤형 레시피 추천 서비스입니다.

---

## 📌 프로젝트 소개
개인 및 가족 단위 사용자를 위한 식자재 관리 애플리케이션입니다.

소비기한 관리, 레시피 추천, 건강정보 기반 음식 추천 기능을 제공합니다.

---

## 🛠 Tech Stack

### Backend
- FastAPI
- MySQL
- SQLAlchemy

### App
- Android
- Java

### ETC
- Firebase FCM
- Retrofit2

---

## ✨ 주요 기능
- 회원가입 / 로그인
- 식자재 CRUD 기능
- 소비기한 알림
- 레시피 추천
- 가족 공유 기능

## 👨‍💻 담당 역할

- 식자재 관리 기능 구현
